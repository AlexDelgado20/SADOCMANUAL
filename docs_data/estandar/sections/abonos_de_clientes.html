Abonos

Despliega la pantalla para realizar abonos de recibos de los clientes.

Botón Abonos

Desde aquí se accede a la función que permite registrar abonos de recibos generados por clientes.

Pantalla de registro de pagos de clientes

En esta pantalla se puede ingresar, anular o consultar los abonos de créditos de los clientes. Si es necesario buscar algún recibo ya registrado, se utiliza la lupa, la cual desplegará los registros creados previamente.

Para generar un nuevo recibo, se debe presionar el botón Nuevo Recibo o utilizar la tecla F5 como atajo de teclado. Una vez hecho esto, se completan los campos correspondientes y se selecciona si el sistema realizará el abono de forma automática o manual, en cuyo caso podrá elegir los recibos específicos a los que se aplicará el abono.

Al finalizar, se debe presionar en Guardar para confirmar el abono. También están disponibles las opciones de Imprimir si solo se desea emitir el recibo, o Anular si se requiere invalidar algún recibo registrado anteriormente.
