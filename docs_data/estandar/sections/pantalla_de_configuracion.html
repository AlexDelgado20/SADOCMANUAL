Pantalla de Configuración

Muestra opciones de configuración para un mejor uso del sistema.

  • Usar nombre corto para factura.
  • Convertir precio Lb a Kg. Convierte el costo de libras a kilos y trabaja con kilos mientras esté seleccionado.
  • Poder vender de contado a clientes CxC. Al ingresar un cliente de crédito, el sistema pregunta si se cobra al contado o a crédito.
  • Agregar CodBar o Ref. al nombre del producto. Agrega esa información en la descripción del producto en la factura.

Pantalla de Configuración

Desde aquí puedes activar o desactivar preferencias y acceder a utilidades como desbloquear RUC, imágenes para pantalla cliente, panel offline, marcado rápido, equipos y sistema.
