Búsqueda

En esta pantalla se consultan los cierres de caja registrados. Selecciona el período de fechas y presiona Buscar; el sistema mostrará la lista de cierres realizados en ese rango para que elijas cuál visualizar.

Vista General

Define las fechas Desde y Hasta y usa el botón Buscar para cargar los cierres disponibles del período. Luego selecciona el cierre deseado para mostrarlo en pantalla.
