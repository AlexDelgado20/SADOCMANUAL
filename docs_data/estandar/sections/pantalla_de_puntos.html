Cliente POS

Este botón despliega una pantalla para consultar o crear clientes de puntos.

Botón Cliente POS

Al presionar este botón se abre la pantalla de clientes, en la cual se pueden consultar, registrar o modificar datos de los clientes que participan en el sistema de puntos.

Pantalla de Datos del Cliente

En esta pantalla se pueden consultar, crear y configurar los clientes que recibirán puntos por cada dólar de compra. Para consultar, se ingresa el ID del cliente y se presiona Enter. Si no se cuenta con el ID, se puede usar el botón Buscar, el cual muestra todos los clientes registrados. También es posible crear un nuevo cliente ingresando su información personal y presionando en Guardar.

Pantalla de Consulta de Puntos

Aquí se muestra el detalle de los puntos obtenidos por el cliente. Se puede visualizar el mes, compras realizadas, puntos obtenidos, así como los pagos realizados y el proceso de canje. En el apartado de Proceso de Cambio se registra el total de puntos a canjear y observaciones adicionales.
