Imágenes para Pantalla de Cliente

Despliega una pantalla de configuración de imágenes en la pantalla del cliente.

Botón Imágenes para Pantalla de Cliente

Permite acceder a la configuración para agregar o gestionar las imágenes que serán mostradas en el monitor del cliente.

Pantalla POS Propaganda

Desde aquí se ingresan las imágenes que se desean mostrar en el monitor del cliente. Para buscar imágenes, presione en el botón Buscar, seleccione desde el explorador de archivos y luego haga clic en Guardar. Para eliminarlas, seleccione la imagen y presione el botón Eliminar en la parte superior derecha de la pantalla.
