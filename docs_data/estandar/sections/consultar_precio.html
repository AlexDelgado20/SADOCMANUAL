Consultar Precio

Este botón abre la pantalla de consulta de precios, diseñada para facilitar la búsqueda sin necesidad de realizar un proceso de facturación en el panel principal.

Botón Consultar Precio

Desde este botón se accede directamente a la función de consulta de precios para productos específicos.

Pantalla de Consulta de Precio

La búsqueda de precios puede hacerse de tres formas: ingresando la descripción, ingresando manualmente el código o usando el escáner de código de barras. Si el ingreso es manual, debe presionar ENTER. Si es por código de barras, el precio se muestra automáticamente. Para cerrar la pantalla, presione la X.
