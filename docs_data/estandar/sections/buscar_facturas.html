Buscar Facturas

Al presionar este botón se despliega una pantalla con las facturas electrónicas realizadas o pendientes. En este formulario se puede reenviar las facturas con impresión, enviar sin imprimir o solo imprimir una copia. También está la opción de enviar todas las facturas pendientes.

Las facturas se pueden buscar por número de documento o por fecha.

Botón Buscar Facturas

Permite abrir la pantalla de gestión de facturas electrónicas, mostrando todas las opciones de reenvío y reimpresión.

Pantalla Buscar Facturas

Aquí se muestran todas las facturas electrónicas realizadas o pendientes. Desde este formulario puede reenviar facturas, imprimir copias, o enviar todas las que aún no se han transmitido.
