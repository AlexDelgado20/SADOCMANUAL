Descuento

Este botón abre la pantalla de descuento, permitiendo aplicar un porcentaje sobre los productos seleccionados.

Botón de Descuento

Al presionar este botón se habilita la pantalla para ingresar el porcentaje del descuento.

Pantalla de Descuento

En esta pantalla se digita el porcentaje a aplicar y se presiona en Aceptar. El descuento se reflejará automáticamente en los ítems seleccionados, siempre y cuando el usuario tenga permisos. Si no cuenta con permisos, existen dos opciones:

  • Aprobación por celular: se envía un mensaje al personal autorizado para aceptar o rechazar el descuento.
  • Validación por clave: un usuario autorizado puede ingresar su clave personal para aprobarlo.
