Marca Rápido

La función de marcación rápida permite acceder de forma ágil a los productos más vendidos o de ventas rápidas. Para ello, debe configurarse previamente desde la pantalla de configuración de marcado rápido.

Acceso a Marca Rápido

Desde el panel principal, al presionar el botón Marca Rápido, el sistema redirige a la pantalla de marcaciones rápidas.

Pantalla de Marcación Rápida

La pantalla cuenta con 300 botones configurables y actualizables. Cada pestaña de opciones incluye 50 botones para asignar productos de ventas rápidas o frecuentes, permitiendo una selección más ágil en el proceso de facturación.
