Suspender

Este botón tiene dos funciones: la primera es suspender automáticamente la transacción en curso, y la segunda es abrir la pantalla de transacciones suspendidas para continuar con el proceso o eliminarlas según sea necesario.

Botón Suspender

Al presionarlo, el sistema guardará la transacción actual como suspendida, para retomarla más tarde o gestionarla en la lista de suspendidas.

Pantalla de Facturas Suspendidas

En esta pantalla puede buscar por número de documento y presionar en Aceptar Documento Interno para continuar la transacción. Para eliminar, seleccione la orden deseada y presione Eliminar Orden de Pedido Seleccionada o bien Eliminar Todas las Ventas Suspendidas.
