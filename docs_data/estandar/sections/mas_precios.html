Más Precios

Este botón habilita la opción de utilizar precios adicionales configurados en el inventario para un producto. En caso de que el producto no tenga precios múltiples, se desplegará una pantalla para ingresar un nuevo precio.

Botón Más Precios

Al presionar este botón, se habilita la opción de seleccionar o ingresar precios adicionales para el producto en proceso de facturación.

Pantalla de Selección de Precio

Si el producto cuenta con más de un precio registrado en el inventario, se mostrará una lista de precios. El usuario selecciona el que desea utilizar en la transacción y presiona en Aceptar. En caso de no tener permisos, el sistema solicitará credenciales de un usuario autorizado.

Pantalla de Ingreso de Precio

Si el producto no cuenta con precios múltiples en el inventario, se desplegará esta pantalla para ingresar manualmente un precio de venta. En caso de no tener permisos, el sistema solicitará la autorización de un usuario con acceso.
