Pantalla de Configuración de Factura Electrónica

Esta pantalla trabaja junto con los datos del cliente. A la izquierda se define el tipo de cliente (consumidor final, gobierno, natural o jurídica) y a la derecha se configura la ubicación del contribuyente/comercio. Además, dispone de las acciones Verificar RUC, Verificar Folios y Buscar Facturas.

Datos del Cliente

Desde aquí se completan el RUC/Cédula, dígito verificador y nombre del cliente. También se pueden agregar observaciones, dirección y teléfono asociados a la factura.

Configuración Electrónica

Defina el tipo de cliente y la ubicación fiscal (provincia, distrito y corregimiento). Utilice los botones para validar el RUC, verificar folios y consultar facturas previamente emitidas.
