Bloqueo

Al presionar el botón Bloqueo, el sistema SADOC POS se bloquea automáticamente hasta que el usuario que estaba usando la caja vuelva a ingresar su clave personal.

Botón de Bloqueo

Este botón garantiza que solo el usuario autorizado pueda continuar con el uso de la caja, brindando mayor seguridad en el punto de venta.

Pantalla de Bloqueo

Una vez activado, se muestra la pantalla de control de acceso. El usuario debe ingresar nuevamente su contraseña personal para desbloquear y seguir operando.
