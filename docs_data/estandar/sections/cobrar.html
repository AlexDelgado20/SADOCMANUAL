F1 – Cobrar

Al presionar este botón o la tecla F1 (atajo de teclado) se habilita la pantalla de cobro.

Botón F1 – Cobrar

Permite acceder a la pantalla de cobro para elegir entre los diferentes métodos de pago disponibles.

Pantalla de Cobro

En esta pantalla puedes escoger el método de pago entre efectivo, tarjeta, cheque, ACH o incluso dividir el pago entre varios métodos. El sistema hace los cálculos automáticamente y al presionar en Cobrar, se imprime la factura y se abre el cajón de dinero.
