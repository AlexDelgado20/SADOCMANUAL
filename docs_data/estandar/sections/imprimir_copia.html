Imprimir Copia

Este botón permite imprimir una copia de la última factura realizada en el sistema, facilitando la entrega de duplicados al cliente si así lo requiere.

Proceso de Impresión

Al presionar este botón, el sistema automáticamente reimprime la última factura registrada. Esta función resulta útil cuando el cliente solicita un comprobante adicional o en caso de que el primer documento haya tenido algún inconveniente de impresión.
