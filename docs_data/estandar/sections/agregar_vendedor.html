Agregar Vendedor

Permite asignar un vendedor de planta cuando la compañía lleva registro de ventas por vendedor. Es útil para calcular bonos o porcentajes por venta, o simplemente para mantener un registro detallado.

Botón Agregar Vendedor

Al presionar este botón, el cursor se moverá automáticamente a la casilla de productos. Allí se debe ingresar el código del vendedor y presionar ENTER.

Vendedor Asignado

Si el registro se realizó correctamente, en la parte superior del panel principal se mostrará el nombre del vendedor asignado.
