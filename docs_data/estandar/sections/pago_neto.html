F2 – Pago Neto

Si el cliente entrega el dinero exacto en efectivo, al presionar este botón o la tecla F2 (atajo de teclado), la transacción se realiza de manera rápida sin necesidad de entrar a la pantalla de cobro.

Botón de Pago Neto

Al utilizar esta opción, el sistema automáticamente genera la factura y abre el cajón de dinero sin pasos adicionales.
