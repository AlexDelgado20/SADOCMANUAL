Historial de Ventas

Muestra un historial de ventas de contado resumido o detallado, con opciones de filtrado por Día actual, Última semana, Último mes, Últimos 6 meses y Último año. También permite exportar a Excel o refacturar la venta seleccionada.

Botón Historial VTA

Permite acceder rápidamente al historial de ventas de contado del sistema.

Pantalla de Historial de Ventas

En esta pantalla se visualiza el historial de ventas de contado, en modo resumido o detallado. Se puede filtrar por rangos de tiempo definidos y acceder al detalle de cada transacción.

Desde aquí también es posible exportar los datos a Excel o refacturar una de las ventas seleccionadas.
