Este botón abre la ventana de pesaje del sistema.

Botón PESA

Permite acceder a la ventana donde se puede leer el peso desde el área de trabajo del sistema.

Pantalla de PESA

En esta pantalla se marcará el peso leído en la báscula conectada al sistema. Después de leer el peso, se mostrará el menú de productos para seleccionar el que corresponde. Automáticamente, el sistema ingresará en el panel de facturación los detalles del producto, el peso registrado y el precio correspondiente.

Nota

OJO: este botón está disponible en la versión de supermercado.
