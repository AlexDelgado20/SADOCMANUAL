Tarjeta de Regalo

Este botón despliega la pantalla para ingresar tarjetas de regalo bajo nombre de la compañía. Primero se ingresa el monto de la tarjeta, luego se escribe o escanea el código de barras. Una vez llenados los campos, presione Guardar. Al estar registrada, la tarjeta de regalo puede ser utilizada como método de pago al cobrar facturas.

Botón de Tarjeta de Regalo

Permite acceder al registro de tarjetas de regalo para su uso como medio de pago en el sistema.

Pantalla de Tarjeta de Regalo

En esta pantalla debe ingresar el monto de la tarjeta y el código de barras (escrito o escaneado). Una vez completado, presione en Guardar para registrar la tarjeta. Esta podrá ser utilizada posteriormente como método de pago.
