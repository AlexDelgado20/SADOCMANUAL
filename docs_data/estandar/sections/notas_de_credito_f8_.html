Pantalla de Notas de Crédito

Se usa para crear notas de crédito a los clientes y para consultar o imprimir sus registros. Puede localizar una venta con la lupa para ver transacciones realizadas o ingresar directamente el número de documento interno.

Boton de Notas de Credito

Flujo Básico

  • Si necesita, use Editar info para modificar RUC, DV, nombre y teléfono del documento. Estas acciones requieren permisos; si no los tiene, el sistema solicitará validación.
  • En la parte superior derecha se dispone de un teclado numérico digital y se muestran los datos del cliente.
  • El botón Ajuste Manual permite ingresar un ítem adicional a la factura, cuando sea necesario.
  • Seleccione los productos a los que se aplicará la nota de crédito y, al finalizar, presione Guardar para generar la nota.
