Pantalla de Inicio de Sesión

En esta pantalla se accede al sistema. Se muestra un panel desplegable con las cajas disponibles en la sucursal, donde el usuario debe seleccionar la asignada. En la parte inferior se encuentran botones numéricos, que se utilizan para ingresar el ID y la contraseña. Cada vez que se ingresa un dato, se debe presionar el botón ENTER para validarlo. Una vez introducidos los datos correctamente, se finaliza dando clic en el botón Entrar.
