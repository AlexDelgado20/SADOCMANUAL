Teclado Numérico

Este teclado numérico es utilizado en cajas con pantallas táctiles donde no se dispone de un teclado físico. Permite ingresar valores de manera rápida y eficiente.

Pantalla de Teclado Numérico

Incluye los números del 0 al 9, doble cero (00), punto decimal, tecla de retroceso y la tecla Enter. En la parte superior se observa un menú que permite acceder a otras pantallas como Electrónica, Opciones y Config.
