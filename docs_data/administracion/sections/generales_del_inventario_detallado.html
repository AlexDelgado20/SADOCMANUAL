Generales del Inventario Detallado

El Catálogo de Inventario Detallado permite registrar y administrar todos los artículos que componen su inventario.

1. Datos Generales del Artículo

En esta sección se registran los datos básicos del artículo que será manejado en el inventario. Estos datos incluyen información como el código, nombre, referencia, cantidades mínimas, y otras características esenciales relacionadas con el artículo.

  • Código: Identificador único del artículo, puede ser generado automáticamente o asignado manualmente.
  • Nombre Largo: Nombre completo del artículo.
  • Nombre Corto: Nombre abreviado del artículo para consultas y reportes.
  • Referencia: Información adicional que identifica el artículo.
  • Cantidad Mínima: Stock mínimo que debe mantenerse en inventario.
  • Pto. Reorden: Cantidad mínima para realizar pedidos de reposición.
  • Costo Actual: Costo actual del artículo en inventario.
  • Unid/Larg/Anch: Dimensiones del artículo en unidades, largo y ancho.
  • Código Barra: Código de barra del artículo (si aplica).

2. Clasificación y Ubicación

Esta sección permite configurar la clasificación del artículo dentro del inventario. Además, se define la ubicación física en la que se almacenará el artículo.

  • Tipo Inv.: Tipo de inventario en el que se clasifica el artículo.
  • Unidad M.: Unidad de medida del artículo.
  • Color: Color asociado al artículo (si aplica).
  • Categoría: Categoría general del artículo.
  • Clase: Clase específica del artículo.
  • Sección: Sección del inventario asignada al artículo.
  • ITBM: Configuración de impuestos aplicables al artículo.
  • Marca: Marca del artículo (si aplica).
  • Línea: Línea en la que se agrupa el artículo.
  • Modelo: Modelo del artículo.
  • Ubicación: Ubicación física donde el artículo será almacenado.
  • Observación: Campo para añadir observaciones adicionales sobre el artículo.

3. Configuración de Precios (Pre. Global)

Esta sección permite configurar los precios globales del artículo, aplicando diferentes niveles de precios según las necesidades de la empresa.

  • Precio1, Precio2, Precio3, Precio4: Diferentes niveles de precios globales para el artículo.
  • Utilidad: Porcentaje de utilidad aplicado a cada nivel de precio.
  • Manual: Opción para configurar los precios globales manualmente.
  • Pedir Clave: Habilitar una clave para ajustes de precios globales.
  • Precio por Margen Ganancia: Define el margen de ganancia en el cálculo del precio global.
  • Borrar: Elimina configuraciones específicas.
  • Borrar Todos: Elimina todas las configuraciones de precios globales.

4. Configuración de Precios por Sucursal (Pre. Sucursal)

Esta sección permite configurar precios específicos para cada sucursal, separando los valores según la ubicación o el manejo de inventario.

  • Sucursal: Nombre de la sucursal.
  • Precio1, Precio2, Precio3, Precio4: Diferentes niveles de precios configurados para la sucursal.
  • Precio Fijo: Opción para fijar el precio sin margen en la sucursal.
  • Precio Margen: Opción para calcular precios según un margen establecido en la sucursal.
  • Actualizar Precio: Botón para actualizar los precios configurados para la sucursal.

5. Configuración de Precios por Mayoristas (Pre. Mayoristas)

En esta sección se configuran precios especiales para mayoristas, ajustando valores específicos para ventas en grandes cantidades.

  • Descripción del Precio: Campo para detallar el tipo de precio configurado para mayoristas.
  • Cantidad Unidad: Cantidad mínima requerida para aplicar precio mayorista.
  • Precio Vta: Precio de venta asignado para mayoristas.
  • Lista de Precios: Muestra la descripción, cantidad, y precio configurado para mayoristas.

6. Impresión de Etiquetas

Esta sección permite configurar la impresión de etiquetas para el artículo, seleccionando la impresora, cantidad de etiquetas, y formato de impresión.

  • Impresora de Etiquetas: Selección de la impresora utilizada para imprimir etiquetas.
  • Cantidad Etiquetas: Número de etiquetas que se imprimirán.
  • Expiración: Configuración del formato de expiración en las etiquetas.
  • Opciones de Formato de Etiqueta:
    • 1 Fila (2.00 x 1.00 Pulgada)
    • 1 Fila (1.25 x 1.00 Pulgada)
    • 2 Filas (1.25 x 1.00 Pulgada)
    • 3 Filas (3.00 x 2.00 Pulgadas - Mayorista)
  • Botón Imprimir Ticket: Permite generar las etiquetas configuradas.

7. Saldo Detallado

Esta sección muestra el saldo detallado del artículo en diferentes bodegas, incluyendo información como serie, tamaño, color, y saldo disponible.

  • Bodega: Nombre de la bodega donde está almacenado el artículo.
  • Serie/Ref: Número de serie o referencia del artículo.
  • Size: Tamaño del artículo.
  • Color: Color asociado al artículo.
  • Saldo: Cantidad de saldo disponible para el artículo.
  • Total: Muestra la suma total del saldo disponible en todas las bodegas.

8. Opciones

En esta sección se configuran opciones adicionales relacionadas con el artículo, que afectan su comportamiento en el inventario y las transacciones.

  • Activo: Indica si el artículo está activo en el inventario.
  • Saldo desglosado: Permite desglosar el saldo del artículo.
  • Permite Descuento: Habilita descuentos en las transacciones del artículo.
  • Agrupar en Documento: Permite agrupar el artículo en un documento.
  • Ventas sin Saldo: Autoriza ventas del artículo aunque no tenga saldo disponible.
  • Pedir en Bodega: Habilita solicitudes del artículo en la bodega.
  • Balanza: Configura el manejo del artículo con balanza.
  • Precio 0: Permite asignar un precio de 0 al artículo.
  • Enviar a Web: Habilita la publicación del artículo en la web.
  • Top Cat. Inf. Caja: Configura la categoría del artículo en información de caja.

9. Controles y Acciones

En esta sección se encuentran los controles principales para interactuar con la configuración del artículo, como guardar, eliminar, salir, entre otros.

  • Verificar Imágenes Corruptas: Opción para verificar si las imágenes asociadas al artículo están corruptas.
  • Reducir Imágenes Captadas: Configuración para reducir el tamaño de las imágenes del artículo.
  • Botón Eliminar: Permite eliminar el artículo configurado.
  • Botón Guardar: Guarda los cambios realizados en la configuración del artículo.
  • Botón Salir: Cierra la ventana de configuración del artículo.
@* Seccion de Registros Contables *@

Registros Contables

Los Registros Contables permiten configurar las cuentas asociadas al manejo de inventarios, compras y ventas dentro del sistema. Además, incluyen opciones para la configuración de factura electrónica, asegurando la correcta integración contable.

1. Cuentas Contables del Inventario

En esta subsección se definen las cuentas contables asociadas al inventario, incluyendo la cuenta de costos y el código arancelario correspondiente.

  • Cuenta Inventario: Define la cuenta contable asociada al inventario.
  • Cuenta Costo: Configura la cuenta contable relacionada con los costos.
  • Código Arancelario: Código utilizado para la identificación arancelaria del artículo.

2. Cuentas Contables de Compras

Configura las cuentas contables relacionadas con las compras, tanto al contado como a crédito, además de las cuentas para devoluciones.

  • Cuenta Compras Contado: Define la cuenta contable para compras realizadas al contado.
  • Cuenta Compras Crédito: Configura la cuenta contable para compras realizadas a crédito.
  • Cuenta Devolución Contado: Configura la cuenta para devoluciones realizadas al contado.
  • Cuenta Devolución Crédito: Configura la cuenta para devoluciones realizadas a crédito.

3. Cuentas Contables de Ventas

Configura las cuentas asociadas a las ventas, incluyendo ventas al contado, ventas a crédito y devoluciones.

  • Cuenta Venta Contado: Define la cuenta contable para ventas realizadas al contado.
  • Cuenta Venta Crédito: Configura la cuenta contable para ventas realizadas a crédito.
  • Cuenta Devolución Venta Contado: Configura la cuenta para devoluciones realizadas al contado.
  • Cuenta Devolución Venta Crédito: Configura la cuenta para devoluciones realizadas a crédito.

4. Balance y Asientos Contables

Visualiza el balance general y registra los asientos contables relacionados con las operaciones de inventario.

  • Asiento: Número identificador del asiento contable.
  • Fecha: Fecha de registro del asiento.
  • Documento: Documento asociado al asiento contable.
  • Descripción: Breve descripción del asiento.
  • Débito: Monto cargado en la cuenta correspondiente.
  • Crédito: Monto abonado en la cuenta correspondiente.
  • Balance: Muestra el balance actual considerando los montos de débito y crédito.

5. Configuración de Factura Electrónica

Configura los parámetros necesarios para la emisión de facturas electrónicas, incluyendo la codificación de bienes y servicios y la unidad de medida fiscal.

  • Codificación Panameña de Bienes y Servicios: Establece el código aplicable a los bienes y servicios según las normas panameñas.
  • Unidad de Medida Fiscal: Define la unidad de medida utilizada para la fiscalización de los artículos.
@* Seccion de Datos Complementarios *@

Datos Complementarios

En esta sección se pueden agregar referencias adicionales del producto seleccionado, datos demográficos para cotizaciones, productos de referencia y configuraciones de productos a descontar en promociones o regalías.

1. Referencias Extras

Esta subsección permite agregar hasta tres referencias adicionales para el producto seleccionado.

  • Referencia Extra 1: Campo para ingresar la primera referencia adicional.
  • Referencia Extra 2: Campo para ingresar la segunda referencia adicional.
  • Referencia Extra 3: Campo para ingresar la tercera referencia adicional.

2. Datos Demográficos para Cotizaciones

En esta subsección se pueden agregar datos relacionados con el producto, como el país de origen, información del fabricante, distribuidor, y detalles de garantía.

  • País de Origen: País donde se produce el artículo.
  • Casa Productora: Nombre del fabricante del producto.
  • Distribuidor: Información del distribuidor del producto.
  • Garantía Meses: Número de meses de garantía para el producto.
  • Garantía Motivo: Descripción de la garantía ofrecida para el producto.
  • Taller o Soporte: Información del taller o soporte técnico asociado al producto.

3. Producto de Referencia

Permite seleccionar un producto de referencia al que pertenece el producto ingresado, convirtiéndose en un subproducto del producto de referencia.

  • Código: Campo para ingresar el código del producto de referencia.

4. Configuración de Productos a Descontar

En esta subsección se configuran los productos que se descontarán al realizar la venta, como parte de promociones o regalías.

  • Id: Identificador único del producto a descontar.
  • Descripción: Breve descripción del producto.
  • Cantidad: Cantidad del producto que será descontada.

Nota: Los productos configurados en esta sección serán descontados automáticamente del saldo de inventario cuando se realice la venta del producto principal.

@* Seccion de Historial de productos *@

Historial del Producto

En esta subsección se puede visualizar el historial de entrada y salida del producto seleccionado, proporcionando información detallada sobre las transacciones realizadas. Esto incluye datos como la fecha, salida, entrada, costo y balance. Es importante destacar que este proceso puede realizarse por sucursal desde el mismo computador, y el registro generado puede exportarse a Excel para análisis y reportes.

1. Historial del movimiento del producto

A continuación, se describen los principales campos disponibles en el historial del producto.

  • Bodega: Indica la bodega donde se encuentra el producto.
  • Id: Identificador único del producto.
  • Producto: Nombre o descripción del producto.
  • Transacción: Tipo de movimiento realizado (entrada o salida).
  • Referencia: Información adicional sobre la transacción.
  • Fecha: Fecha exacta de la transacción.
  • Balance: Saldo del producto después de la transacción.
  • Entrada: Cantidad de producto ingresada.
  • Salida: Cantidad de producto retirada.
  • Costo: Costo asociado al producto en la transacción.
@* Seccion de Historial de modificaciones *@

Historial de modificaciones

En la pantalla de Historial de Modificaciones, puedes visualizar una bitácora detallada de los cambios realizados en los productos, incluyendo información sobre el usuario que realizó la modificación, la fecha en que se efectuó y el equipo desde el cual se hizo el cambio.

1. Historial de modificaciones

La pantalla de Historial de Modificaciones proporciona una bitácora detallada que muestra:

  • Usuario: Indica qué usuario realizó una modificación.
  • Fecha: Muestra el momento exacto donde se realizo la modificación
  • Equipo: Identifica desde qué equipo se realizó la modificación (campo "PC").
@* Sección de Importación de Precio *@

Importación de Precio

En la pantalla de Importación de Precio, el usuario puede actualizar los precios desde un archivo Excel. Para hacerlo, debe tener el archivo Excel listo, especificar el nombre de la hoja en el campo correspondiente y hacer clic en el botón importar. Después de importar, debe presionar el botón de actualizar precios para que los cambios se apliquen automáticamente.

1. Proceso de Importación de Precio

La pantalla de Importación de Precio permite realizar las siguientes acciones:

  • Importar: Cargar los datos desde un archivo Excel especificando el nombre de la hoja.
  • Actualizar precios: Aplicar los cambios automáticamente en el sistema.
  • Actualizar por código de barra: Actualizar precios basados en el código de barra del producto.
  • Actualizar por referencia: Si no hay código de barra, utilizar la referencia única del producto.
