Registro de Cierre Contable

El registro de cierre contable es utilizado para realizar cierres al finalizar cada mes o cada período fiscal. El proceso puede ser mensual o anual, y permite asegurar la correcta contabilización y reporte de los saldos contables.

Cierre Mensual

El cierre mensual se realiza seleccionando la fecha correspondiente al mes de operación y presionando el botón "Cerrar Mes". Esta acción genera las transacciones necesarias para el cierre contable del mes.

Detalle de Cierre Mensual

Aquí se visualizan las transacciones de cierre, tipo de movimiento, valores de débito y crédito, y el total de la operación.

Acciones de Cierre Mensual

Utilice los siguientes botones para completar el proceso de cierre mensual: cerrar mes, exportar a XLS o cerrar pantalla.

Cierre Anual

El cierre anual se realiza al finalizar el período fiscal, normalmente el 31 de diciembre. Se debe verificar el período de operación, seleccionar la fecha para el asiento de cierre anual y el departamento correspondiente.

Datos y Reglas de Cierre Anual

El cierre anual genera dos asientos fiscales que no se estiman en el reporte de Balance General de Cuentas, pero sí se consideran en la columna de Balance Anterior.

Acciones de Cierre Anual

Utilice los siguientes botones para completar el proceso de cierre anual: cerrar período fiscal, eliminar cierre fiscal o cerrar pantalla.
