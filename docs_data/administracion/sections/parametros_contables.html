Parámetros Contables

En esta pantalla configuras las cuentas y parámetros relacionados con clientes, proveedores y aspectos generales y de base de datos contable. Los cambios afectan la contabilidad y operaciones financieras del sistema.

1. Clientes

Esta sección permite configurar las cuentas relacionadas con las ventas, cobros y notas de crédito para clientes.

  • Cuentas de ventas.
  • Descuentos.
  • Cuenta de ITBMS.
  • Departamento de débito y crédito para transacciones específicas.
  • Cuenta para cobros.
  • Cuenta para notas de crédito.

2. Proveedor y Gasto

En esta sección se definen las cuentas contables utilizadas para registrar las compras, pagos y obligaciones con proveedores.

  • Cuenta por pagar.
  • Cuenta de compras.
  • Cuenta de ITBMS.
  • Departamento asociado.
  • Cuenta para pagos.

3. Generales

Aquí se configuran parámetros globales como el período fiscal, el mes operativo y cuentas clave para el cierre de año.

  • Establece el período fiscal a trabajar.
  • Define el mes operativo.
  • Cuentas para generar utilidades al cierre fiscal o anual.
  • Cuenta de capital.
  • Cuenta para sobrantes y faltantes para registrar diferencias en ventas.

4. Base de Datos

Esta sección permite definir las bases de datos relacionadas con operaciones financieras del sistema, como ahorro o préstamos.

  • Configura el nombre de la base de datos de ahorros y préstamos.
  • Otras bases de datos relacionadas.
