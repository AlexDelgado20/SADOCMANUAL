Solicitudes

Muestra el historial de solicitudes (requerimientos / pedidos internos) dentro de un rango de fechas. Permite filtrar por categoría, marca, bodega y también localizar directamente un producto específico. Desde esta vista puede exportar o imprimir la información para seguimiento y control.

Vista General

Nota

  • Defina primero el rango de fechas para limitar la consulta antes de añadir más filtros.
  • Use el campo Producto para auditar rápidamente las solicitudes de un ítem crítico.
  • Combine Categoría y Bodega para analizar la demanda interna por línea en una sucursal específica.
  • Revise las cantidades elevadas junto con el Monto para detectar solicitudes que requieren aprobación adicional.
  • Si no aparecen resultados verifique que la bodega y la fecha correspondan al origen real de las solicitudes.
  • Exportar los datos permite comparar solicitudes versus consumos o compras posteriores.
  • Un Monto inusualmente alto puede deberse a actualizaciones recientes de costos; confirme la vigencia del costo unitario.
