Definir Precio de Venta Fijo por Inventario

Esta opción es utilizada cuando los artículos del inventario ya tienen estipulados precios fijos de venta y no se trabaja en base a márgenes de ganancia. Ejemplos incluyen productos como gaseosas, picaritas, dulces, entre otros.

1. Filtros

Utilice los siguientes filtros para buscar los artículos:

  • Categoría: Seleccione la categoría del artículo.
  • Clase: Seleccione la clase del artículo.
  • Marca: Seleccione la marca del artículo.
  • Código: Rango de código inicial y final para filtrar.
  • Longitud del Código: Permite establecer la longitud del código.
  • Actualizar Costo: Opción para actualizar los costos del inventario.
  • Actualizar Código de Barra: Opción para ajustar los códigos de barra de los artículos.

2. Listado de Artículos

Esta tabla muestra los artículos del inventario que tienen precios fijos de venta. Las columnas de color verde son editables y deben procesarse con cautela.

Nota: Si configura el precio en base a márgenes de ganancia, los artículos que se encuentren en el presente listado y cuyo precio sea mayor a 0, solo quedarán con el precio que usted asigne aquí.

3. Acciones

Utilice los siguientes botones para gestionar la configuración:

  • Buscar Inv: Permite buscar artículos en el inventario.
  • Guardar: Almacena los cambios realizados.
  • Salir de Pantalla: Cierra la ventana actual.
