Transacciones

Desde aquí puede ver el historial de transacciones del inventario filtrando por fecha, categoría, marca y bodega, además de consultar directamente un ítem (Items) o un documento específico. Permite imprimir o exportar a Excel los resultados obtenidos.

Vista General

Nota

  • Defina primero el rango de fechas para reducir el volumen de datos antes de añadir filtros secundarios.
  • Utilice Items cuando requiera validar rápidamente todos los movimientos de un producto específico.
  • El campo Document es útil para auditar un ajuste o transferencia particular.
  • Si no obtiene resultados verifique que la bodega seleccionada tenga movimientos en el período.
  • Active Copia de Histórico solo cuando necesite revisar datos migrados o archivados; esto puede ampliar el tiempo de consulta.
  • Exporte después de aplicar todos los filtros para minimizar el tamaño del archivo Excel.
