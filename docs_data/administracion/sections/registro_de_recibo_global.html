Registro de Recibo Global

Esta pantalla permite registrar un Recibo Global para aplicar un cobro a distintos documentos del cliente. El número de recibo es consecutivo automático (preconfigurado), por lo que el usuario sólo ingresa el código del cliente y presiona Enter para cargar sus datos.

1. Datos Generales

  • Recibo: consecutivo automático (Nuevo Recibo F5).
  • Fecha: día del cobro.
  • Cliente: ingrese / busque y Enter para cargar.
  • Personal: usuario o ejecutivo asociado (opcional).

2. Parámetros Contables

  • Cuenta Caja: cuenta donde ingresa el dinero.
  • Cuenta del Recibo / Documento / Abono: asignación puntual.
  • Procesar Asignación: registra la línea a aplicar.
  • Generar A.D. por Corrección: asiento de ajuste (si aplica).

3. Detalle de Aplicaciones

  • Tipo (C/P/A): C = Contabilidad / CxC, P = Préstamo, A = Ahorro.
  • Cuenta / Código / Nombre: identifica destino contable o producto.
  • Documento: seleccionar cuidadosamente para evitar duplicidad.
  • Saldo / Deuda y Monto / Abono: comparación y valor aplicado.
  • Total inferior: suma de los montos agregados.
  • Observación general: nota global del recibo.
  • Nota informativa: no mezclar recibos de Apartados con otros tipos; abonos de Apartados generan comprobante fiscal, los demás no fiscal.

4. Acciones

  • Guardar: valida e invoca detalle de forma de pago.
  • Imprimir: emite el comprobante final.
  • Anular: revierte antes de usos posteriores (según permisos).
  • Salir: cierra la pantalla.
