DGI: Informe 1027

Desde aquí puede consultar y preparar el Informe 1027 para la DGI, seleccionando el periodo deseado y exportando el resultado en Excel. El archivo generado está listo para ser enviado a la institución.

Vista General

Nota

  • Solo se incluirán documentos que tengan asignado un número de Factura Fiscal.
  • Revise cuidadosamente los datos antes de exportar; puede transformar ventas a contado si es necesario.
  • El archivo exportado cumple con los requisitos de la DGI para el informe 1027.
