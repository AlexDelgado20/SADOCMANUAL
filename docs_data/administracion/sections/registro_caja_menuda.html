Registro de Caja Menuda

Esta pantalla se utiliza para el registro de los comprobantes generados en la Caja Menuda. Aquí se muestra el número de comprobante, la persona que lo utilizó, la cuenta contable a la que se aplicará el gasto, el monto cargado y el monto de ITBMS pagado si corresponde. Además, permite el registro de adelantos de salario que algunas empresas manejan mediante esta mecánica. Al finalizar el registro, presiona el botón guardar.

Datos del Comprobante

  • Id Compro: Número identificador del comprobante.
  • Cuenta CM: Cuenta de Caja Menuda asociada.
  • Fecha: Fecha del comprobante.
  • Personal: Persona responsable del gasto.
  • Tipo de Desembolso: Selecciona entre Gasto, Cargo CxC o Abono/Cancelación CxP.

Detalle del Gasto

Aquí se detallan los gastos aplicados: cuenta, departamento, descripción, datos del proveedor, tipo de persona, concepto, compra de bienes y servicios, factura, monto, ITBMS y total.

Acciones Disponibles

Utiliza los siguientes botones para gestionar los comprobantes de Caja Menuda: recálculo, eliminar, anular, guardar o salir.
