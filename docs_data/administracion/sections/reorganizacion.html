Reorganización del Inventario

Formulario con 4 pestañas para reorganizar: Configuración Actual, Nueva Configuración, Eliminar Configuraciones y Eliminar Productos. Se incluye además la vista del listado.

1. Configuración Actual

  • Filtros: Marca, Categoría, Clase, Sección.
  • Filtrar Por: Nombre Inventario, Referencia, Nombre Proveedor o CodBarra.
  • Se usa para localizar productos y tomar referencia antes de aplicar cambios.

2. Nueva Configuración

  • Seleccionar nueva Marca, Categoría, Clase y Sección para asignar a los productos.
  • Asignar ITBMS y Cant. Mínima si corresponde.
  • Botones: Quitar de Web, Activar a Web, Actualizar Config., Actualizar Impuesto, Actualizar Cant.Min, Salir.

3. Eliminar Configuraciones

  • Seleccionar únicamente un registro (Marca o Categoría o Clase o Sección).
  • Si no está asignado a productos se eliminará permanentemente.
  • Botones: Eliminar, Salir. Nota de advertencia en pantalla.

4. Eliminar Productos

  • Elimina los productos listados siempre que cumplan: sin ventas realizadas y sin compras realizadas.
  • Botones: Eliminar, Salir. Nota muestra condiciones.

5. Listado de Productos

  • Grilla con columnas: Código, CodBarra, Producto, Marca, Categoría, Clase, Sección.
  • Se actualiza según los filtros aplicados en Configuración Actual.

Nota

  • Verificar siempre antes de eliminar configuraciones o productos para evitar pérdidas irreversibles.
