Traspaso de Mercancía

En esta pantalla es donde vamos a revisar solicitudes de inventario existentes para esta bodega y/o generar un nuevo traspaso entre bodegas. Si existe una solicitud previa puede aceptarse (referenciándola); de lo contrario se crea un traspaso directo (Nuevo Traspaso F5) y se completa la información.

1. Encabezado

  • Número: consecutivo del traspaso (Nuevo Traspaso F5).
  • Solicitud: referencia a la solicitud de inventario (si proviene de una petición previa).
  • Fecha: fecha de registro del traspaso.

2. Bodegas y Parámetros

  • Salida: bodega origen (de donde se descuenta mercancía).
  • Entrada: bodega destino (donde ingresa la mercancía).
  • Recontabilizar esta Bodega (enlaces): opción para recalcular o validar saldos/costos (según configuración interna).
  • Cta Crédito Salida / Cuenta Débito Entr: cuentas contables asociadas al movimiento (pueden mostrarse deshabilitadas si el sistema las determina automáticamente).
  • Manejo (%): porcentaje que se aplicará a todos los productos de la bodega entrante (mensaje en pantalla indica aplicación global). Ajustar la interpretación según política (markup, manejo, merma, etc.).
  • Seleccione para usar Scanner: habilita captura por lector de códigos.
  • Seleccione para ver Imágenes: muestra imágenes asociadas a los artículos.

3. Productos

  • Producto de Salida: CódigoSal, DescripcionSal, UNI, Marca, CantidadSa.
  • Producto de Entrada: CodigoEnt, DescripcionEnt, UNI, Marca, CantidadE, CostoEnt, Total.
  • Existencia: stock disponible de la bodega de salida para el producto seleccionado.
  • Totales: sumatoria de cantidades y montos (parte inferior de la grilla).
  • Se pueden registrar múltiples líneas; cada línea asocia salida y entrada (en transferencias 1:1 o según equivalencias definidas).
  • Obvervacion: Campo para comentarios generales sobre el traspaso (motivo, referencia interna, incidencias).

5. Nota

  • Advertencia: los traspasos no se pueden anular, se debe revisar cuidadosamente antes de contabilizar.
  • En caso de duda consultar al administrador del sistema.

6. Acciones

  • Contabilizar: ejecuta el traspaso, descuenta existencias de la bodega de salida y las ingresa en la bodega de entrada (afecta inventario y contabilidad).
  • Ver Imágenes: muestra imágenes asociadas a los productos (catálogo/visor).
  • Imprimir: genera documento de soporte del traspaso.
  • Guardar: registra el traspaso en estado pendiente (sin contabilizar) para revisión.
  • Salir: cierra la pantalla.
