Listado de Ventas Vs. Costos

Esta sección muestra un informe comparativo entre las ventas y los costos del inventario. Permite filtrar por período, categoría, clase, marca y bodega, así como por tipo de venta (contado, crédito o ambos).

Vista General

Defina el rango de fechas, seleccione los filtros deseados y el tipo de venta. Luego presione Visualizar para generar el reporte de comparación entre ventas y costos.
