Listado de Ventas

Esta sección muestra el registro de ventas realizadas en un período específico. Permite consultar por cuenta, descripción, bodega y generar reportes de ventas de forma diaria, mensual o por clasificación de producto.

Vista General

Seleccione el rango de fechas y configure los filtros deseados para generar el listado. Puede optar por un informe diario, mensual o clasificado por producto.
