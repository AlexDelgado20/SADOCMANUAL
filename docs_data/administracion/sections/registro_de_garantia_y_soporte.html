Registro de Soporte Técnico y Garantía

Esta pantalla permite crear, consultar y actualizar trámites de soporte o garantía. Busque un IdRegistro existente o genere uno nuevo (F5). Complete datos del cliente y de los productos involucrados, registre eventos/estatus y al finalizar actualice (Cerrar) o imprima el documento.

1. Datos Generales

  • IdRegistro: buscar o Nuevo Documento (F5).
  • Fecha: día del ingreso o actualización.

2. Datos del Cliente

  • Cédula / RUC: identifica y carga Nombre, Dirección, Teléfono.
  • N° Factura Sistema: vínculo de compra para validar garantía.
  • Observación General: resumen del problema o solicitud.

3. Productos y Estatus

  • Agregar productos manualmente (requiere aprobación): Código, Descripción, Marca, Unidad, Cantidad, Referencia.
  • DGAR: días de garantía restantes (si aplica).
  • Estatus: registrar avance mediante eventos (cruz / + para añadir estado con fecha y nota).

4. Acciones

  • Crear Evento al Producto: añade estatus con fecha y nota.
  • Imprimir: genera comprobante del trámite.
  • Cerrar: guarda/actualiza el registro.
  • Salir: cierra la pantalla.
