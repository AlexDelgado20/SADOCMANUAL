Listado de Movimiento Categórico

Esta sección permite registrar, organizar y consultar los movimientos categóricos del inventario. Está dividido en diferentes niveles y opciones para definir categorías, asignar artículos y generar reportes.

Definición de 1° Nivel

Cree y administre categorías principales para clasificar el inventario. Registre descripciones y guarde los datos para usarlos en el sistema.

Definición de 2° Nivel

Establezca subcategorías relacionadas con las categorías del primer nivel para detallar aún más la clasificación de los artículos.

Asignación de Valores

Asigne artículos a los subtítulos seleccionados para consolidar la información categórica del inventario.

Reporte

Genere el informe de movimientos categóricos seleccionando el rango de fechas y el grupo de primer nivel.
