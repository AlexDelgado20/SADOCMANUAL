Registro de Consignaciones

Emita documentos de consignación cuando el cobro dependa de la venta posterior del cliente. No genera cuenta por cobrar hasta su cierre; solo afecta inventario según los artículos entregados.

1. Encabezado y datos del cliente

  • Número: Consecutivo del documento. “Click() para Nuevo Documento” crea uno nuevo.
  • Cliente Id: Identificador del cliente.
  • RUC/CID y DV: Identificación fiscal.
  • Nombre, Dirección, Teléfono: Datos de contacto del cliente.

2. Opciones

  • Término Venta: Condición de referencia para la entrega.
  • Personal Venta: Vendedor responsable.
  • Cuenta: Cuenta de caja asociada.
  • Bodega: Origen del inventario consignado.
  • Facturas: Campo de referencia si aplica relación con facturas.

3. Detalle de artículos consignados

  • Ingrese IdArt para agregar líneas.
  • Columnas: Código, Referencia, Descripción, Unidad, Cantidad, Precio, Descuento, Total.
  • Líneas: Contador de ítems agregados.
  • Obs. General: Notas del documento.
  • Totales: Sub-Total, Descuento, ITBM y Total.

4. Acciones

  • Imprimir (F4): Genera el formato de consignación.
  • Guardar (F1): Registra el documento (sin CxC).
  • Salir: Cierra la pantalla.
  • Flujo sugerido: Completar datos → Agregar artículos → Guardar → Imprimir.
