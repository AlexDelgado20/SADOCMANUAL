Definición de Presupuesto Anual

En esta pantalla se asigna el presupuesto anual y mensual a las diferentes cuentas del sistema. Por ejemplo, una empresa de transporte puede definir aquí el presupuesto para combustible de cada mes del año. Este proceso permite tener una referencia clara del dinero presupuestado y el dinero realmente gastado al final del año.
Para utilizar el formulario: haz clic en nuevo registro, selecciona la fecha y el periodo al que se asignará el presupuesto, agrega una observación y guarda el registro.

Datos del Presupuesto

  • Registro: Número identificador del presupuesto.
  • Fecha: Fecha de creación del presupuesto anual.
  • Periodo: Periodo al que se asignará el presupuesto.
  • Observación: Comentarios o notas adicionales sobre el presupuesto.
