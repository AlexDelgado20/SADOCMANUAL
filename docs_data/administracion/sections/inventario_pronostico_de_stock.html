Consulta de Pronóstico de Stock

Muestra un registro de los productos junto con sus órdenes de compra: órdenes emitidas, cantidades recibidas y pendientes de recepción. Facilita visualizar el abastecimiento próximo y anticipar faltantes o sobre-inventarios según el estado de las órdenes abiertas.

Vista General

Nota

  • Para agilizar la carga, refine primero por Categoría o Marca antes de listar un rango amplio de productos.
  • Si un producto no muestra ODC pero debería, valide que la orden esté aprobada y no anulada.
  • El cálculo de Pendiente asume que devoluciones ya están descontadas del campo Recibido (confirme la lógica interna).
  • Exportar permite combinar estos datos con reportes de ventas promedio para construir un cálculo de cobertura futura (Stock Disponible = Cantidad + Pendiente).
  • Use Observacion para registrar estados manuales que ayuden a logística (ej. “Esperando despacho”, “Reprogramado”).
