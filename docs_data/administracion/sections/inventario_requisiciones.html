Requisiciones

Muestra las requisiciones realizadas en un período específico. Permite consultar tanto requisiciones de salida de inventario como de entrada (por devoluciones, ajustes o retornos), filtrando por personal solicitante/responsable y por bodega. Los resultados pueden exportarse o imprimirse para control y seguimiento.

Vista General

Nota

  • Seleccione primero el rango de fechas para limitar la cantidad de registros antes de refinar por Personal o Bodega.
  • Use el Tipo para separar análisis de consumos (Salida) versus retornos o ajustes (Entrada).
  • Revisar Observacion ayuda a identificar el propósito de la requisición (proyectos, mantenimiento, producción, etc.).
  • Un Monto alto en conjunto con muchos TItems puede requerir aprobación adicional según políticas internas.
  • Si Status muestra documentos pendientes, valide que se hayan completado los procesos de aprobación o contabilización.
  • Exporte los datos para conciliarlos con reportes de consumo de centros de costo o con ajustes contables.
  • Ante discrepancias revise si consultó el tipo correcto (Salida vs Entrada) y el rango de fechas esperado.
