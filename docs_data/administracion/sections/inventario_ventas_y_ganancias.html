Ventas y Ganancias

Muestra el registro de las ventas con sus respectivas ganancias dentro de un rango de fechas. Puede visualizar la información en forma Detallada o Resumida según la necesidad de análisis. Permite filtrar por categoría, clase y bodega, además de exportar los resultados.

Vista General

Nota

  • Use primero el rango de fechas adecuado para limitar el volumen de datos y agilizar la carga.
  • La pestaña Consulta Detallada es ideal para auditorías o revisión de clientes y vendedores; la Resumida para reuniones ejecutivas.
  • CantVta menos CantNCr refleja la cantidad neta vendida; verifique esta relación cuando las devoluciones sean elevadas.
  • Si la Ganancia luce anómala revise si existen descuentos atípicos o notas de crédito posteriores.
  • Exporte cada vista por separado si necesita comparar el nivel agregado versus el detallado.
  • Para performance, evite consultar períodos muy extensos en modo Detallado; use antes el Resumido y luego profundice.
  • Si alguna columna aparece en blanco valide permisos del usuario o configuración del reporte (por ejemplo ocultar costos).
