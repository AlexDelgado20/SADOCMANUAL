Solicitud de Inventario

En este apartado se van a generar solicitudes de productos a otras bodegas: crear documento, definir fechas, bodega solicitada, solicitante, detallar artículos y guardar/imprimir.

1. Encabezado

  • Número: consecutivo interno (Nueva Solicitud F5 para generar).
  • F. Realizada: fecha en que se registra la solicitud.
  • F. Despacho: fecha deseada para que se despache/envíe la mercancía.
  • Bod. Solicita: bodega que está solicitando los productos (destino).
  • Per. Solicita: usuario / persona que hace la solicitud.
  • Observación: nota breve justificando o describiendo la necesidad.

2. Detalle

  • Ingreso de los productos requeridos.
  • Columnas: Línea, Código, Descripción, Unidad, Cantidad, Costo (si se muestra), Total.
  • Cantidad: unidades solicitadas a la bodega origen.
  • Costo / Total: referencial (puede usarse para valoración interna o dejarse vacío si solo se controla cantidad).
  • Fila Total inferior: suma de cantidades y montos (si se manejan costos).

3. Acciones

  • Imprimir C/ Costo: genera reporte mostrando valores de costo.
  • Imprimir S/ Costo: genera reporte sin valores de costo (solo cantidades).
  • Imprimir Catálogo: versión de impresión tipo listado/catálogo de ítems solicitados.
  • Cerrar: (según configuración) marca la solicitud como cerrada/finalizada (no editable) o confirma su envío.
  • Guardar: registra la solicitud en estado pendiente o lista para gestión posterior.
  • Salir: cierra la pantalla sin crear nueva acción.
