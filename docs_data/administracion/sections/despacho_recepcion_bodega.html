Despacho / Recepción de Bodega

Pantalla para registrar productos que salen (despacho) o entran (recepción) al inventario indicando la bodega destino. A diferencia de la Requisición, aquí no es necesaria la creación previa de una cuenta para ejecutar la operación.

1. Encabezado

  • Número: consecutivo interno (Nuevo Documento F5 para generar).
  • Fecha: fecha del registro (despacho o recepción).
  • Operación: tipo de movimiento (ej. Despacho, Recepción, Ajuste).
  • Bodega Entr.: bodega que recibe (en recepciones) o se utiliza según la operación configurada.
  • Manejo (%): porcentaje extra al costo (mensaje indica que aplica solo a las Recepciones).
  • Obs. General: comentario o nota descriptiva del movimiento.

2. Cuentas Configuradas

  • Cta Dr: cuenta contable de débito asociada (definida por la operación, no editable aquí).
  • Cta Cr: cuenta contable de crédito asociada (definida por la operación).
  • Se muestran solo como referencia; no se requiere crear una cuenta manualmente en esta pantalla.

3. Detalle

  • Ingreso de líneas de artículos que salen o entran.
  • Columnas: Código, CodBar, Descripción, Unidad, Marca, Cantidad, Costo (referencial / para recepciones), Total.
  • Totales inferiores: suma de cantidades y montos.
  • El costo de salida no se modifica (ver nota de seguridad).

4. Acciones

  • Contabilizar: aplica el movimiento al inventario y registra el asiento contable (usa las cuentas configuradas).
  • Doc. Costo: impresión / reporte con detalle y costos.
  • Documento: impresión / reporte general (puede omitirse costo según formato).
  • Guardar: almacena en estado pendiente (sin contabilizar aún).
  • Salir: cierra la pantalla.

5. Notas

  • Por seguridad, no se modifica el costo de salida.
  • Por seguridad, si cambia de bodega se borrará todos los productos del documento para ingresarlos de nuevo.
