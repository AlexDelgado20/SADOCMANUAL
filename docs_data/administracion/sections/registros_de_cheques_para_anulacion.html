Datos para Anulación

  • Banco: Seleccione el banco por código.
  • Cuenta: Cuenta contable asociada al banco.
  • No. Cheque: Ingrese el número exacto del cheque que desea anular.
  • Proveedor: Campo informativo; suele mostrarse un proveedor modelo.
  • Nombre: Referencia del proveedor/beneficiario asociado al cheque.
