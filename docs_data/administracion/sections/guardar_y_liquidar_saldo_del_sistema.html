Guardar y Liquidar Saldo del Sistema

Proceso para crear (apertura) y posteriormente liquidar (cierre) los saldos del inventario del sistema con base en el conteo físico. Se crea y guarda el documento antes de registrar los saldos físicos; luego de cargarlos se liquida y se generan los ajustes e informes.

1. Apertura de Liquidación (Campos)

  • Código: identificador del documento de liquidación.
  • Descripción: texto descriptivo del proceso.
  • Fecha: fecha de apertura.
  • Bodega: bodega a liquidar.

2. Indicadores y Opciones de Informe

  • Valor Inv. Neutro: valor sin diferencia (base).
  • Ganancia: valor positivo que resulte de diferencias a favor.
  • Perdida: valor negativo por faltantes.
  • Informe de Inventario para captar saldos: listado para anotar cantidades físicas.
  • Informe de Inventario para verificar saldos general: listado para revisión global.

3. Acciones de Apertura

  • Imprimir: genera el informe seleccionado (captar o verificar) antes del conteo físico.
  • Guardar: registra el documento de apertura para reutilizarlo en el cierre.
  • Salir: cierra sin ejecutar liquidación.

4. Cierre de Liquidación

  • Checkbox: Cambiar Cuenta de Ganancias y Pérdidas de Inventario por la de Costo de Inventario (redirige el asiento si se marca).
  • Botón Liquidar y Realizar Ajustes: ajusta saldos, crea requisiciones de ajuste (entrada y salida) y genera el asiento contable.
  • Nota inferior: indica que al liquidar se ajustan los saldos y se generan los asientos respectivos.

5. Notas

  • El documento debe estar guardado antes de capturar los saldos físicos.
  • Verificar diferencias (ganancia / pérdida) antes de liquidar.
  • Conservar informes impresos como respaldo del conteo físico.
  • Después de liquidar, ajustes posteriores se manejan con transacciones de ajuste independientes.
