Listado de Flujo de Caja

Este reporte muestra el flujo de ingresos y egresos, ya sea diario o en un período específico. Para que funcione correctamente, primero debes configurar las cuentas de ingresos, luego las de gastos y, por último, las cuentas de efectivo que quieras incluir. Registra cada cuenta en el campo “Cuenta”, presiona Enter para agregarla y repite el proceso según el tipo. Una vez visibles todas las cuentas, guarda la configuración para generar un reporte preciso.
