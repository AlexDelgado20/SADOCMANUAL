Ventas y Devoluciones

Consulta las ventas y devoluciones realizadas en un rango de fechas. Muestra datos detallados: caja, tipo de documento, número de cierre, personal / supervisor, fecha y hora, documento/fiscal, forma de pago, cliente, subtotal, impuestos y total.

Vista General

Nota

  • Seleccione Transacción = Todos, Solo Ventas o Solo Devoluciones según el análisis requerido.
  • Use Caja para auditar un cierre específico; doble clic en la fila abre la factura (mensaje en rojo).
  • El botón Graficar genera un resumen visual del período filtrado; Exportar produce un archivo para análisis externo.
