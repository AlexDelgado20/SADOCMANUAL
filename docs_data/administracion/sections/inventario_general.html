General

Muestra el listado de artículos del inventario. Permite filtrar por proveedor, categoría, clase, marca, bodega y nombre del inventario, además de localizar productos según si tienen o no margen asignado. Se puede exportar o imprimir el resultado.

Vista General

Nota

  • Use Nombre Inv. para localizar rápidamente un producto específico antes de exportar.
  • Si necesita revisar artículos sin parámetros comerciales utilice Sin Margen Asignado.
  • Combine Proveedor y Categoría para evaluar surtido por línea.
  • Revise Minimo para detectar productos por debajo del nivel recomendado.
  • Exporte solo después de aplicar todos los filtros para reducir el tamaño del archivo.
