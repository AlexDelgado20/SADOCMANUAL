Datos de Apertura

  • Caja: Caja operativa en la que se trabajará.
  • Cajero: Usuario cajero asignado a la caja.
  • Autorizado: Supervisor que valida la apertura.
  • Clave: Clave de autorización del supervisor.
  • Monto Ap.: Efectivo inicial con el que abre la caja.

Formulario de selección de caja, cajero, autorizado, clave y monto de apertura.
