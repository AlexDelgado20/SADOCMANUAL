Registro de Facturas – Productos Detallada

Descripción breve de cada campo visible en las imágenes.

1. Encabezado

  • Proveedor: código del proveedor.
  • Factura: número de la factura del proveedor.
  • O.d.C.: número de orden de compra (si aplica).
  • Fecha: fecha del documento.
  • Bodega Trab.: bodega donde ingresará la mercancía.
  • Departamento: departamento relacionado.
  • Ter. Venta: términos de venta.
  • Obser.: observación general.
  • Usar Costo Cero (0): opción para costos cero (promos).
  • Usar descuento por unidad: aplica descuento por cada unidad.
  • Usar descuento total del producto: opción para usar un solo descuento total.

2. Costeo de Gasto

  • Tabla: lista de gastos asociados (Proveedor, Factura, Monto).
  • Total: suma de los montos listados.
  • + Agregar Gasto registrado: enlace para añadir un gasto.

3. Detalle de Productos

  • Filas de productos con columnas:
  • Códig: código del producto.
  • CodBar: código de barras.
  • Referenc: referencia.
  • Nombre: nombre del producto.
  • Cuenta: cuenta asociada.
  • Fecha Exp: fecha de expiración (si aplica).
  • Cantidad: unidades recibidas.
  • Costo: costo del producto.
  • PrecioVta: precio de venta.
  • Flete: costo de flete por producto.
  • CostoV: costo variable adicional.
  • Desc: descuento de la línea.
  • Total: total de la línea.
  • Importar: botón para importar desde archivo.
  • Costo Actual: costo vigente.
  • Costo Futuro: costo a aplicar más adelante.
  • Saldo: saldo mostrado del producto.
  • Click Cambio Precio de Venta al Cliente: enlace para ajustar precios.
  • N° Trazabilidad: número de lote / serie.
  • Total Unidades: suma de cantidades.
  • Total Flete: suma de fletes.
  • Total Varios: otros totales varios.
  • SUB-TOTAL: subtotal de líneas.
  • DESCUENTO: total de descuentos.
  • ITBM: impuesto.
  • TOTAL: total final.

4. Acciones

  • Anular: anula el documento.
  • Ajustar: permite ajustes.
  • Contabilizar: procesa el documento contablemente.
  • Eliminar: elimina líneas / documento (según estado).
  • Documento: imprime el documento.
  • Etiquetas: imprime etiquetas.
  • Precios Vta: abre cambio de precios.
  • Guardar: guarda sin contabilizar.
  • Salir: cierra la pantalla.
