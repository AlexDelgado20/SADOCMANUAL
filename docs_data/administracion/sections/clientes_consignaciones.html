Consignaciones

Desde aquí podemos ver y unificar las consignaciones registradas. Se puede filtrar por bodega, tipo de cliente y estado (activos o cerrados) además de un rango de período. Permite unificar varias consignaciones bajo un código, exportar o cerrar la vista.

Vista General

Nota

  • Para unificar: indique en Id Doc. el código destino, marque en la columna Unificar las consignaciones y luego presione Unificar.
  • No marque a la vez Mostrar solo Activos y Mostrar solo Cerrados si necesita ver ambos estados.
  • Use el período más corto posible para agilizar la consulta antes de exportar.
  • Verifique totales después de unificar para confirmar la consolidación correcta.
  • Utilice doble clic para revisar el detalle antes de incluir una consignación en la unificación.
