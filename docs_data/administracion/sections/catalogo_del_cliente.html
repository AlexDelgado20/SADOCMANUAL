Catálogo del Cliente

Luego de clasificar los clientes, debemos registrar los datos generales de cada uno de ellos, como Nombre, cédula, dirección y otros datos de interés para la empresa, así como el Certificado de Operación (utilizado únicamente en las Cooperativas de transporte). Si el cliente está activo, se selecciona en la casilla correspondiente; en caso de estar inactivo, se deja en blanco.

1. Datos Generales del Cliente

En esta sección se registran los datos básicos del cliente, incluyendo información como código, nombre, RUC/DV, contacto, dirección y otros datos relevantes.

2. Cuentas y Precios

Aquí se muestran las opciones para configurar las cuentas contables asociadas al cliente y establecer precios de venta predeterminados.

3. Estado y Credenciales

Esta imagen detalla las configuraciones de estado del cliente (Activo, ITBMS, Validación de saldo, etc.) y las credenciales para el portal web de compras.

4. Opciones

En esta sección se incluyen las opciones para reasignar un nuevo ID al cliente, guardar cambios, eliminar registros, o salir del formulario.

Bitácoras

En esta sección se pueden visualizar y añadir observaciones sobre el cliente seleccionado en Generales del Cliente.

1. Observaciones

Área donde puedes escribir y registrar observaciones importantes acerca del cliente.

2. Historial de Observaciones

Visualiza el historial de observaciones realizadas, incluyendo detalles como la fecha y el contenido registrado.

Notas sin Aplicar

Esta sección muestra las notas de crédito sin aplicar del cliente seleccionado, con información como ID, nombre, NCR, cuenta y saldo.

1. Detalle de Notas

Visualiza las notas de crédito pendientes de aplicación, incluyendo ID, Nombre, NCR, cuenta y saldo del cliente.

Historial Contable

En esta sección se puede visualizar el historial de transacciones y cuentas detalladamente. También permite exportar esta información hacia Excel si es necesario.

1. Transacciones y Cuentas

Visualiza las transacciones realizadas en las cuentas del cliente y selecciona las cuentas asociadas.

2. Detalle de Transacciones

Visualiza un listado detallado de las transacciones contables realizadas.

Historial Auxiliar

En esta sección se pueden visualizar las cuentas y saldos pendientes del cliente, así como el registro detallado de facturación y pagos realizados.

1. Ventas a Crédito

Área para aprobar las ventas a crédito realizadas por el cliente.

2. Facturación y Pagos

Registro detallado de las facturas emitidas y los pagos realizados por el cliente.
