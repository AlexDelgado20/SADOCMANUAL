Histórico de Modificaciones

Muestra el historial de modificaciones realizadas a los productos dentro de un rango de fechas. Permite filtrar por código (o buscar un producto específico) y visualiza la descripción del hecho y el usuario responsable. Pensado para auditorías, control de cambios, trazabilidad y soporte.

Vista General

Nota

  • Utilice rangos de fechas acotados (ej. últimos 7 o 30 días) para un desempeño óptimo.
  • Si un cambio no aparece, valide que el usuario tenga permisos para ver auditorías y que el evento esté configurado para registrarse.
  • El detalle de la columna Descripción depende de la granularidad configurada (puede listar campos alterados o un resumen genérico).
  • Para análisis forense, exporte inmediatamente tras detectar una incidencia, antes de que se purguen registros antiguos (si aplica política de retención).
  • Considere establecer un flujo de aprobación para cambios sensibles basado en la información que aquí se visualiza.
