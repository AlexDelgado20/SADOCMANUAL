Registro de Cotización

Procedimiento para emitir cotizaciones a clientes: el número puede ser automático (o seguir un consecutivo preimpreso), se identifica al cliente para cargar sus generales, se define vigencia, tiempos y términos, se agregan productos con sus cantidades y precios, y finalmente se guarda e imprime el documento.

1. Encabezado y datos del cliente

  • Número: Consecutivo de la cotización. “Nueva Cotización (F5)” crea un registro nuevo.
  • Cliente Id: Identificador del cliente; completa automáticamente Nombre/Teléfono y generales.
  • Nombre/Tel: Razón social o nombre y teléfono de contacto.
  • R.U.C./D.V.: Identificación fiscal del cliente.
  • Email: Correo de contacto para enviar la cotización.
  • Dirección: Domicilio o dirección comercial.
  • Personal Venta: Asesor o vendedor responsable.

2. Vigencia y condiciones

  • Válido hasta: Fecha límite en que aplica la cotización.
  • Tiempo Entrega: Días estimados para entregar.
  • Término Venta: Condición de pago (p. ej., Contado, Crédito).
  • Opciones:
    • Precio Mayorista según la Cantidad.
    • Incluir Código Interno de Inventario en la cotización.
    • Cambio de Precio (autorizado).

3. Detalle de productos

  • Ingrese Código, Cantidad y Precio Unitario; presione Enter para agregar líneas.
  • Columnas: Código, CodBarra, Descripción, Marca, Unidad, Cantidad, Precio, Descuento, Total.
  • Obs. General: Observaciones para el cliente.
  • Totales: Sub-Total, ITBM, Descuento y TOTAL.
  • Accesos laterales: Histórico, Precio Gerencia, Cubicaje, Inventario, Precio Manual.

4. Acciones

  • Guardar: Registra la cotización.
  • Imprimir / PDF / Imprimir 2: Genera el documento para entregar al cliente.
  • Detalle: Vista/edición del detalle según formato.
  • Salir: Cierra la pantalla.
  • Flujo sugerido: Completar datos → agregar productos → Guardar → Imprimir/PDF.
