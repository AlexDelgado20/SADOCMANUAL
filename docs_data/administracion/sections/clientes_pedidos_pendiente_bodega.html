Pedidos Pendientes en Bodega

Desde aquí podemos ver los pedidos pendientes de entrega en el sistema y realizar su entrega. También se puede abrir el historial de pedidos para tener la referencia completa antes de procesar. Incluye impresión en formato 8.5x11 o Comanda.

Nota

  • Use Historial antes de entregar para validar si existen notas previas o entregas parciales.
  • La columna Días ayuda a priorizar pedidos con mayor antigüedad.
  • Revise Cant. Pend. antes de presionar Entregar para evitar procesar cantidades incorrectas.
  • Imprima la Comanda si requiere un documento interno para preparación en bodega.
