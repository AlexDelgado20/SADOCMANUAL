Parámetros Asistentes

Esta sección permite configurar los datos necesarios para que el sistema pueda enviar un correo automático, ya sea a clientes y/o proveedores desde una cuenta designada.

1. Configuración del Remitente

Esta sección te permite establecer los datos necesarios para que el sistema pueda enviar correos automáticos desde una cuenta designada. Aquí configuras el correo del remitente y los detalles técnicos requeridos para el envío.

  • Email: Dirección de correo electrónico que será utilizada como remitente en los mensajes automáticos.
  • Contraseña: Clave de acceso para autenticarse en el servidor de correo del remitente.
  • Servidor Mail: Servicio de correo saliente (por ejemplo, smtp.gmail.com), seleccionable según tu proveedor.
  • Srv/Port: Puerto de conexión y dirección específica del servidor de correo (por ejemplo, 587 para SMTP).
  • Encabezado: Texto personalizado que aparecerá en el encabezado de los correos enviados.

2. Email de Envios

En esta sección se define a qué personal se enviarán los correos automáticos generados por el sistema.
Para recibir estos correos, debe configurar en los datos generales del personal un email válido en la casilla correspondiente.

  • Los correos serán enviados a los destinatarios que usted configure.
  • Debe ingresar el email en la ficha de datos generales del personal para que reciba las notificaciones.
  • Solo los usuarios con un correo válido recibirán estos mensajes automáticos.
