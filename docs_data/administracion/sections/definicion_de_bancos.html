Definición de Bancos

En esta opción configuramos los datos generales de cada uno de los bancos que la empresa utiliza para mantener cuentas bancarias. Por cada entidad bancaria definida, se debe asignar la cuenta contable correspondiente, un prefijo que identifique la cuenta y la secuencia del cheque disponible. Incluso si el banco no tiene chequera, pero se mantiene una cuenta, debe definirse en esta sección.

Datos Generales del Banco

  • Código: Identificación única del banco.
  • Descripción: Nombre del banco.
  • Teléfono: Número de contacto del banco.
  • Proveedor: Proveedor asociado, si aplica.

Secuencia de Cheques

En este apartado se asigna la cuenta contable, el prefijo y la secuencia del cheque para cada cuenta bancaria. Esto es fundamental para la gestión y control de los cheques emitidos por cada banco.

Acciones Disponibles

Utilice los siguientes botones para gestionar los bancos: eliminar, guardar o salir.
