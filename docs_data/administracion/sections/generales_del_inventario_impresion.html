Generales del Inventario - Impresión

Este menú está creado para la fácil impresión de detalles sobre algún producto de su inventario. Los usuarios pueden ingresar información clave y generar un reporte en formato imprimible.

1. Detalles del Producto

Complete los campos necesarios con información del producto para preparar la impresión:

  • Código: Identificador único del producto.
  • Nombre Largo: Descripción extensa del producto.
  • Referencia: Detalle adicional para identificar el producto.
  • Código Barra: Código de barras del producto (si aplica).
  • Observación: Notas o comentarios adicionales sobre el producto.
  • Cantidad: Número de unidades del producto.
  • Expira (Opcional): Fecha de expiración del producto (si aplica).

2. Acciones Disponibles

Utilice los botones para realizar las siguientes acciones:

  • IMPRIMIR: Genera un reporte detallado para impresión.
  • LIMPIAR: Reinicia el formulario para agregar nueva información.
  • SALIR: Cierra el menú y regresa al panel principal.
