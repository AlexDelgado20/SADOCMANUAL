Listado de Margen de Ganancia

Esta sección muestra el margen de ganancia del inventario o de productos específicos. Permite filtrar por categoría, clase, sección, marca, unidad, ITBM, ubicación, bodega y estado (activos, inactivos o ambos). Además, se puede generar el reporte en formato detallado o como un listado de costos resumido.

Vista General

Configure los filtros según sus necesidades, seleccione el tipo de listado (detallado o costos resumido) y use el botón Visualizar para generar el informe.
