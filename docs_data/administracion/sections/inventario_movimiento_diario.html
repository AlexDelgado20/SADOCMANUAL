Muestra el movimiento del inventario por bodega en una fecha específica. Si tiene más de una bodega o sucursal puede consultar cualquiera de ellas, generar un reporte y exportar los datos para análisis adicional.

Nota

  • Seleccione primero la Bodega para evitar tiempos de consulta innecesarios.
  • Use la fecha para revisar variaciones puntuales (por ejemplo auditorías de un día específico).
  • El Balance refleja el neto del día (Entrada menos Salida) y ayuda a detectar ajustes inusuales.
  • Genere el Reporte antes de Exportar si requiere una vista resumida para impresión.
  • Si aparecen valores vacíos verifique que existan movimientos registrados en esa fecha y bodega.
