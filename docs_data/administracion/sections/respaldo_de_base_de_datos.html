Respaldo de Base de Datos

Esta sección permite realizar copias de seguridad de la información del sistema, guardando los datos en una ubicación seleccionada. El respaldo asegura que puedas recuperar tu información en caso de fallos, daños o imprevistos, almacenando la copia en otra computadora o dispositivo externo como CD, DVD o USB.

1. Registro de Licencia de Auditores Externos

Permite registrar el serial de la licencia para auditores externos, necesario para validar y acceder a procesos específicos del sistema.

  • Ingrese el serial de la licencia en la casilla correspondiente.
  • El serial garantiza el acceso y validez de la auditoría externa.
  • Verifique que la licencia esté activa para evitar inconvenientes al realizar respaldos o auditorías.

2. Operación a Realizar

Selecciona la acción que deseas realizar sobre la base de datos del sistema: copia de seguridad o creación de una nueva compañía.

  • Copia de Seguridad: Realiza un respaldo de la base de datos actual, guardando una copia de la información.
  • Crear una Nueva Compañía: Inicia el proceso para crear una nueva base de datos para otra compañía.
  • Base de Datos: Muestra el nombre de la base de datos seleccionada para la operación y permite verificar antes de proceder.
