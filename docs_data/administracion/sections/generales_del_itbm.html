Generales del ITBM

En esta sección configuramos los tipos de impuestos y el monto a aplicar en cada una de las transacciones en el sistema SADOC.

  • Código: identificador único para el tipo de impuesto.
  • Descripción: nombre o detalle del impuesto configurado.
  • Monto: porcentaje o cantidad aplicada al impuesto.
  • Cuenta: código de cuenta asociada para grabado y no grabado.

El sistema tiene predeterminado el 7%, el 10%, impuesto sobre licor, 15% impuesto sobre cigarrillos, y usted puede crear en caso de existir otro porcentaje o impuesto aplicado a su empresa. Recuerde que los montos asignados aquí tienen efecto sobre todo el inventario existe en el sistema. Pues el cargo de ITBMS se ejecuta sobre los productos y no sobre el cliente o proveedor.
