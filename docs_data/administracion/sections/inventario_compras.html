Compras

Muestra las compras realizadas por cada artículo dentro de un rango de fechas. Se puede filtrar por categoría, bodega o directamente por un ítem específico. Permite exportar o imprimir la información para análisis o control interno.

Vista General

Nota

  • Defina primero el rango de fechas para limitar el volumen de datos antes de aplicar más filtros.
  • Use el campo Items para auditar rápidamente el historial reciente de compra de un solo producto.
  • Combine Categoría y Bodega para evaluar abastecimiento por línea en una sucursal específica.
  • Compare Costo y Valor entre períodos exportando diferentes rangos y conciliando los totales.
  • Si Saldo muestra valores inesperados verifique si el reporte está configurado para mostrar saldo post-compra o solo la cantidad comprada.
  • Exportar después de filtrar garantiza archivos más livianos y fáciles de analizar.
