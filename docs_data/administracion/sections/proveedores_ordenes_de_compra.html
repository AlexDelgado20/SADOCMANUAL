Órdenes de Compra

Muestra las órdenes de compra detalladamente dentro del rango de fechas indicado.

1. Filtro

  • Fecha / Al: rango de fechas para listar órdenes de compra.

2. Detalle

  • Columnas: Documento, Fecha, Proveedor, AfeInv (indica si afecta inventario), InvCod, Descripción, Cantidad, Costo, Monto.
  • Total: muestra suma de montos (y/o cantidades según configuración) al pie.

3. Acciones

  • Buscar: carga las órdenes según el rango de fechas.
  • Exportar: genera archivo (por ejemplo Excel) con lo mostrado.
  • Imprimir: envía el listado actual a impresión.
  • Cerrar: cierra la consulta.
