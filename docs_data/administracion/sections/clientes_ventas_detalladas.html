Ventas Detalladas

Vista detallada de las ventas por cliente. Permite filtrar por rango de fecha (o ignorar la fecha), categoría, marca, bodega, personal y nombre del cliente. Incluye opciones para agrupar o resumir resultados, limitar a ventas de contado o solo clientes retenedores de ITBMS. Puede exportar o imprimir según necesite.

Vista General

Nota

  • Use Agrupar Resultado o Sumarizar Resultado para reducir filas antes de exportar.
  • Desmarque Considerar Fecha si necesita un histórico completo del cliente.
  • Combine filtros (Cliente + Categoría o Marca) para análisis más rápido.
