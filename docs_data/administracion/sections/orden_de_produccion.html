Orden de Producción

En esta pantalla vamos a registrar la fabricación de productos: datos del documento, cliente, producto a fabricar, materias primas, costos y contabilización.

1. Encabezado

  • Número: consecutivo interno de la orden (Nuevo Documento F5 para generar).
  • Fecha: fecha de creación de la orden.
  • Depart: departamento / área de inventario que gestiona la producción.

2. Datos del Cliente

  • Cotización: referencia aprobada (opcional) que origina la orden.
  • Id Cliente: código del cliente (permitido crear / buscar).
  • Nombre / Teléfono / Dirección: datos de contacto y entrega.
  • F. Inicial: fecha de inicio del proceso productivo.
  • F. Entrega: fecha comprometida de entrega del producto terminado.

3. Producto a Fabricar

  • Bodega: bodega donde se registrará el producto terminado.
  • Código / Descripción: identificación del producto a fabricar.
  • Solicitado: cantidad requerida.
  • Entregado: cantidad ya producida y entregada (se actualiza a medida que se completa).
  • Personal: responsable / cuadrilla asignada.
  • Costo Uni: costo estimado unitario (resultado de MP + costos/gastos varios / mano de obra).

4. Materia Prima

  • Ingresar cada insumo requerido para producir el producto final.
  • Columnas habituales: Bodega, InvCódigo, Descripción, Unidad, Cantidad, Costo, Total.
  • El Total de la sección suma el costo de todos los insumos consumidos.
  • Obs. General (parte inferior): notas del proceso, incidencias, ajustes o instrucciones para planta.

5. Costos y Gastos Varios

  • Registrar costos indirectos: mano de obra adicional, energía, empaques, transporte interno, servicios.
  • Columnas típicas: Tipo, Cuenta (contable), Nombre (detalle), Monto.
  • El total de esta pestaña se suma al costo de la Materia Prima para calcular el costo unitario final.
  • Revisar que las cuentas imputadas correspondan al plan contable de producción.

6. Acciones

  • Contabilizar: procesa la orden, descuenta la materia prima del inventario y genera (o incrementa) el producto terminado según cantidades entregadas.
  • Imprimir: genera documento / hoja de trabajo para planta o control interno.
  • Guardar: guarda en estado pendiente (sin afectar inventario ni contabilidad aún).
  • Salir: cierra la pantalla sin realizar cambios adicionales.
