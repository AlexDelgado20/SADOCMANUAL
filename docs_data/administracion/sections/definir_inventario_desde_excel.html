Definir Inventario desde Excel

En esta pantalla se puede cargar un inventario desde Excel hacia el sistema Sadoc. Es necesario seguir los comentarios y seleccionar las opciones que se desean subir al software. Posteriormente, se elige la sucursal para cargar o modificar el inventario, se busca la ubicación del archivo Excel en el equipo, y se procesa la información.

1. Opciones Principales

Estas opciones permiten configurar cómo los datos serán procesados desde el archivo Excel:

  • Usar Referencia como llave principal: Utiliza la referencia como clave única en ausencia del código de barras.
  • Generar Código de Barras: Genera un código de barras para los productos que no lo tengan.
  • Cambiar Precio, Cambiar Costo, Sumar Cantidad: Ajusta precios, costos o cantidades en base al código de barra.
  • Usar Precio Mayorista: Configura precios mayoristas para los productos.

2. Opciones de Columnas

Seleccione las columnas que tendrá el archivo Excel. Es obligatorio incluir las columnas de Descripción, Categoría e Impuesto:

  • BarCode: Código de barras del producto.
  • Descripción: Breve descripción del producto (marcado por defecto).
  • Categoría: Clasificación del producto.
  • Clase, Marca, Color: Detalles adicionales del producto.
  • Impuesto: Tipo de impuesto aplicable.
  • Costo: Costos asociados al producto.
  • Cantidad: Cantidad disponible.
  • Cantidad_Caja: Cantidad por caja.
  • Referencias: Referencia1, Referencia2, Referencia3.
  • Precios: PrecioVta, PrecioVta2, PrecioVta3, PrecioVta4.

Nota: La hoja de Excel debe llamarse "INVENTARIO" y las columnas deben llamarse igual a las opciones seleccionadas.

3. Selección de Sucursal, Área de Visualización y Botones

Seleccione la bodega o sucursal donde se cargarán las cantidades:

  • Selector de Sucursal: Campo desplegable para seleccionar la bodega o sucursal.
  • Área de Visualización: Espacio vacío donde se mostrarán los datos cargados del archivo Excel.
  • Indicadores: IbInicial e IbTotal muestran posibles valores relacionados con el inventario inicial y total.
  • Botón Buscar Archivo de Excel: Permite localizar el archivo Excel en el equipo.
  • Botón Salir: Cierra la pantalla actual.
  • Botón Procesar: Inicia el procesamiento de los datos del archivo Excel.
