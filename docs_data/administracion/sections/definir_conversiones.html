Definir Conversiones

La conversión permite asignar acciones automáticas al programa para gestionar productos agotados en una unidad de medida, utilizando otro producto del mismo tipo pero con una unidad de medida diferente. Por ejemplo, si se agotan las botellas de un desinfectante pero hay galones disponibles, el programa tomará un galón y lo convertirá en litros, ajustando el saldo de ambas unidades para realizar la venta.

1. Configuración de Conversión

Complete los siguientes campos para definir las conversiones:

  • Código: Identificador del producto.
  • Descripción: Breve descripción del producto.
  • U. Medida: Unidad de medida asociada al producto.

2. Detalles de Conversión

Proporcione los detalles necesarios para la conversión:

  • Cantidad: Cantidad que será convertida.
  • Prioridad: Nivel de prioridad para realizar la conversión.

Utilice los botones disponibles para agregar la conversión a la lista, guardar los cambios realizados, o salir de la pantalla actual.
