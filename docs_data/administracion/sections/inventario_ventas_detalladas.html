Ventas Detalladas

Muestra las ventas realizadas por cada artículo dentro de un rango de fechas. Permite filtrar por categoría, marca, bodega o directamente consultar un ítem específico. Desde aquí puede exportar o imprimir la información para control y análisis.

Vista General

Nota

  • Comience definiendo el rango de fechas para reducir el volumen de datos antes de aplicar otros filtros.
  • Utilice el campo Items para auditar rápidamente las ventas de un producto concreto.
  • Combine Categoría y Marca para analizar desempeño de una línea específica.
  • Exportar después de filtrar genera archivos más pequeños y manejables.
  • Compare diferentes períodos exportando rangos separados y evaluando variaciones en Valor total.
