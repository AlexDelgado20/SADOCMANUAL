Registro de Notas de Crédito

Se utiliza para devolver mercancía vendida: se crea la Nota de Crédito, se vincula (cuando aplique) a la factura original, se ingresan las cantidades devueltas y su costo; luego se puede aplicar el saldo generado a facturas pendientes del mismo cliente.

1. Encabezado

  • Número: consecutivo automático (Nuevo Documento F5).
  • Doc. Interno / Factura Fiscal: referencias a la factura o documento base.
  • Cliente Id: completa Nombre, Email, Dirección y DV.
  • Use la búsqueda para localizar la factura a la que se aplicará la devolución.

2. Opciones

  • Término Venta y Personal Venta (para trazabilidad).
  • Confeccionado: fecha de emisión de la nota.
  • Bodega: desde donde retorna el inventario.
  • Cuenta (CtaVta) y bandera Vta Sin Itbm si corresponde reversar sin impuesto.

3. Fact. Electrónica

  • Tipo Cliente y Contribuyente para timbrado.
  • Provincia, Distrito, Corregimiento: datos requeridos.
  • Verificar RUC antes de emitir para evitar rechazos.

4. Aplicación

  • Lista facturas con saldo para aplicar la Nota de Crédito.
  • Saldo Disponible muestra el crédito generado.
  • Seleccione facturas (anteriores a la fecha indicada) y defina Abono.
  • Aplicar confirma; Cancelar descarta selección.
  • Mensaje de advertencia: ejecutar con responsabilidad.
