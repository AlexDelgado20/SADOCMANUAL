Registro de Depósitos

Esta pantalla permite registrar los depósitos efectuados en el banco. El sistema asigna automáticamente un número de secuencia para cada depósito. Luego se selecciona la cuenta según el banco, se coloca una breve descripción y el monto depositado, y finalmente se presiona el botón de guardar.
En caso de cometerse un error en el registro, el depósito puede ser anulado, siempre y cuando la corrección se realice dentro del mismo mes de registro.

Datos del Depósito

  • Depósito: Número identificador del depósito.
  • Fecha Ing.: Fecha en la que se efectuó el depósito.
  • Cta Caja: Cuenta de caja general asociada.
  • Banco: Banco en el que se realizó el depósito.
  • Cuenta: Cuenta bancaria asociada.
  • Descripción: Breve detalle del depósito.
  • Monto: Valor depositado.

Acciones Disponibles

Utilice los siguientes botones para gestionar el registro de depósitos: anular, contabilizar, guardar o salir.
