Muestra detalladamente el historial de ventas a crédito por cliente y bodega. Desde esta pantalla se pueden actualizar los días y letras de crédito de los clientes y exportar la información para análisis externo.

Nota

  • Utilice el filtro Días de Crédito para localizar clientes que requieren ajuste.
  • Antes de usar Act. día Cre seleccione únicamente los registros que será necesario modificar.
  • Revise UltPago para evaluar puntualidad antes de ampliar días o cambiar la letra de crédito.
  • Exporte después de aplicar todos los filtros para reducir el tamaño del archivo.
  • Si un cliente no aparece verifique que tenga ventas a crédito registradas en la bodega seleccionada.
