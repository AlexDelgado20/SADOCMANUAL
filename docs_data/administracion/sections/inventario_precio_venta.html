Precio Ventas

Desde aquí puede buscar por marca, categoría y producto el ítem de inventario del cual desea conocer el precio de venta. Permite además limitar la consulta solo a artículos con saldo y exportar el resultado.

Vista General

Nota

  • Active Ver Solo con Saldos para evitar mostrar productos agotados.
  • Use parte del nombre o referencia en Dato a Buscar para filtrar rápidamente.
  • Si no aparece un producto verifique que coincida con la marca o categoría seleccionada.
  • Exportar después de aplicar todos los filtros reduce el tamaño del archivo.
  • Utilice doble clic para validar visualmente el producto antes de informar el precio al cliente.
