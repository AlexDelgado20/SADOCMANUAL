Listado de Códigos de Tiquetes

Esta sección permite generar e imprimir códigos de tiquetes para productos del inventario. Se pueden obtener en base al saldo actual, facturas de proveedor, requisiciones o por una cantidad manual definida.

Artículo Específico

Seleccione una bodega y un artículo específico para generar los códigos de tiquetes.

Grupo de Productos

También puede generar los tiquetes por grupo de productos filtrando por categoría, clase, sección o marca.
