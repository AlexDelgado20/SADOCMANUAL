Compras a Proveedores

Muestra las compras realizadas a cada proveedor específico según filtros seleccionados.

1. Filtros (Fechas y Datos Básicos)

  • Fecha / Al: rango para limitar las compras consultadas.
  • Bodega: filtra por la bodega donde se registró la compra.
  • Categoría: restringe a productos de una categoría.
  • Producto: selección directa de un producto (icono búsqueda).
  • Proveedor: selección de un proveedor específico (icono búsqueda).

2. Filtros (Clase y Marca)

  • Clase: filtra por la clase del producto dentro de la categoría.
  • Marca: restringe a productos de una marca específica.
@* Seccion Detalle de compras *@

3. Detalle de Compras

  • Columnas: Proveedor, Factura, Fecha, Código, BarCode, Referencia, Descripción, Unidad, Cantidad, PrecioV (precio de venta vigente o referencial), Costo, CostoFlet (costo de flete), CostoImp (costo de impuesto), Descuento (si se muestra), y otros que el sistema tenga configurados.
  • Total General: suma de campos numéricos principales (por ejemplo cantidades y costos) al pie.
  • La grilla se llena después de ejecutar la búsqueda con los filtros aplicados.
@* seccion acciones de compra *@

4. Acciones

  • Buscar: ejecuta la consulta con los filtros definidos.
  • Exportar: genera un archivo (ej. Excel) con las compras listadas.
  • Imprimir: envía el listado actual a formato de impresión.
  • Cerrar: sale de la consulta.
