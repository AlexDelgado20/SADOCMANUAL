Introducción y Configuración del Sistema

SADOCPOS – Restaurante es una solución integral diseñada para optimizar la gestión de ventas, facturación y servicios en establecimientos gastronómicos. Esta versión se especializa en el flujo de trabajo dinámico de un restaurante, incluyendo la toma de órdenes, gestión de mesas y múltiples métodos de pago.

Pantalla de Acceso

Al iniciar el sistema, el usuario debe identificarse mediante su ID y contraseña. Es fundamental contar con los permisos adecuados para acceder a las funciones críticas de caja y administración.

Parámetros del Sistema

Antes de la operación diaria, se deben configurar los parámetros base:

  • Conexión: Configuración del servidor y la base de datos centralizada.
  • Sucursal: Asignación del punto de venta a una sucursal específica.
  • Estación: Ajustes locales como impresión de logo, tipo de facturación (TFHKA, etc.) y activación de modos adicionales como monitor de clientes o sistema de puntos.
