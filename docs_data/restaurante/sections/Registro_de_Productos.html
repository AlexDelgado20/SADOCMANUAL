REGISTRO DE PRODUCTOS

Antes de entrar en esta pantalla el sistema le va a pedir validación, después de ingresar las credenciales va poder ingresar. para ingresar un producto se presiona en el signo de mas (+), llena los campos como se indica, si aún no tiene categorías se ingresa un nombre de categoría, por ejemplo, entradas, bebidas, almuerzos, etc., le agrega un color a la categoría, selecciona entre las opciones de funcionalidad del producto, Si ya tiene categorías creadas las puede buscar en la flecha que está a un costado de donde ingresa el nombre de categoría, también puede cambiar el nombre en el icono de la agenda. Si tiene código de barras se ingresa, si no va a utilizar puede dejar en blanco y si quiere crear uno presiona en el icono de código de barras. En el parte inferior va a encontrar un menú de pantallas en los cuales va a terminar de configurar el producto.

  • Precio global – ingresar el precio por sucursal del producto.
  • Materia prima – ingresar materia prima del producto si corresponde.
  • Acompañamiento o promo – ingresar los productos que salen como acompañamiento o promoción al momento de comprar el que se está registrando.
  • Condición – ingresar si hay alguna condición para la transacción con este producto.
  • Tamaños o porciones – ingresar el nombre de las porciones si aplica y precio de estas.
  • Extras – ingresar descripción de las extras y precios si aplican.
  • Productos Vta. – ingresar productos que pueden ser vendidos con el que se esta configurando.
  • Fact. Elec. – se ingresa el código de los productos para la factura electrónica que por lo general es restaurantes y catering. Se asigna la unidad de medida del producto si corresponde.
