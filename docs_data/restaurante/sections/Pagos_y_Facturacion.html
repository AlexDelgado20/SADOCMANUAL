MÓDULO DE COBROS

Pantalla de cobro: Esta pantalla se utiliza para cobrar, ingresar detalles del cliente e ingresar datos de la factura electrónica.

Sección de cobro:

Se escoge el método de pago e incluye el registro para propina.

Sección de teclado numérico:

Digitación de montos entregados por el cliente.

Datos del cliente:

Nombre, país, cédula, dirección y correo electrónico. Si es un cliente ya registrado, al digitar la cédula el sistema llena automáticamente los datos.
