SADOCPOS – RESTAURANTE

SADOCPOS – RESTAURANTE, es una de las versiones del software y está orientada al manejo correcto y eficiente de las ventas en restaurantes a la hora de facturación y realización de órdenes. Dentro este sistema podrá encontrar diferentes módulos como, pre-cuenta, comensal, cobros, mesa. El SadocPos, también maneja otras pantallas con otros módulos necesarios para su usabilidad, como la pantalla de opciones donde se encuentran configuraciones y funciones extras del sistema.

El modulo de cobros, se utiliza para identificar el método de pago del cliente y la impresión de su factura.

En la pantalla opciones, se muestran diferentes funciones en forma de módulos que mejoran la usabilidad y agregan funciones al software. Algunas de las opciones que se muestran son, cierre de caja, notas de crédito, mesero, imprimir copia, reservaciones, clientes vip, entre otras.

El modulo de mesa muestra las mesas y secciones del restaurante.

Todos los módulos, opciones y funciones mencionadas y las demás no mencionadas que contemplan el sistema trabajan en conjunto para hacer más cómoda la atención al cliente y el trabajo del personal en los aspectos que competen a un restaurante como lo son, facturación y toma de pedidos.
